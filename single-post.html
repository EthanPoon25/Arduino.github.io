15 minute Arduino Introduction Video

Home / Learn how to use the Arduino

This video provides an overview of Arduinos, explaining what they are, what they can do, and how to set them up for various projects. Arduinos are circuit boards with microcontrollers that can run small software programs. The origins and background of Arduino is also discussed

Comment: "One of the best tutorials of ANY kind you'll see here. No anoying music, no glitches with mics or camera, clearly spoken, planed ahead so he knows what the hell he wants to say before he says it. Outstanding. Not a wasted moment."

The video covers the Arduino hardware, including the different types of Arduinos and third-party variants. It also introduces Arduino "shields," which are additional circuit boards that expand the capabilities of the main Arduino board. Next, the video discusses the Arduino software development environment, which is beginner-friendly compared to traditional microcontroller programming. The Arduino software is compatible with Windows, Mac, and Linux, and it simplifies the process of uploading code to the Arduino via a USB cable.

Next, the video explores the Arduino Uno, a popular Arduino board for beginners. The speaker explains the components on the board, including microcontrollers, crystal resonators, USB communication chips, power sources, and pin connectors. The various pins on the Arduino Uno are explained, such as power pins, digital input/output pins, analog input pins, and pulse width modulation pins. The software aspect is covered, including downloading and installing the Arduino software, setting up the correct Arduino board type, and selecting the appropriate serial port for communication with the Arduino. The speaker demonstrates a basic program example using analog input to measure voltage with a potentiometer and displays the results on the computer using the serial monitor and plotter. Another example shown is a simple LED dimmer circuit controlled by the Arduino.
