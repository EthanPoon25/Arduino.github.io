Creative Arduino Projects

Home / Learn how to use the Arduino

In case you want a quick overview of all the projects, they are listed below. However, I strong advise you to watch the videos as they briefly touch on how each product is made and used.

  1. Pocket-Sized Computer: A custom PCB, tiny OLED, and Arduino Nano are used to create a pocket-sized computer with features like a calculator, stopwatch, games, phone book, and calendar.
  2. Robotic Bartender: This project involves a smart bartender controlled via Bluetooth, using a paddle to pour drinks based on user input.
  3. DIY Gimbal: A DIY gimbal is created using a two-axis FPV BDC assembly, batteries, a joystick, and an Arduino. The gimbal provides stability and range of motion for high-quality video recording.
  4. Boxing Glove Controller: A unique controller is made using a boxing glove and an Arduino. The glove allows the player to activate abilities in the game Final Fantasy 14 by throwing punches.
  5. Rotary Shortcut Keyboard: A macro keyboard with a rotary encoder is built to provide shortcut keys for various tasks, including changing tabs and media controls.
  6. Programmable Firework Controller: An Arduino Nano is used to create a firework controller with 50 outputs, enabling remote control of fireworks for a display.
  7. Arduino-Based Smart Glasses: These smart glasses, controlled via a smartphone app, display the date, time, and notifications. They are powered by a rechargeable battery.
  8. One-Axis Self-Balancing Device: A self-balancing robot uses an MPU-6050 sensor and an Arduino to maintain balance on a thin beam or spikes.
  9. Animatronic Eye: This project involves building animatronic eyes that can move realistically using servos and Arduino control.
  10. Arduino Gauge: A custom gauge is created using an Arduino Uno and a 16x2 character display to monitor any desired parameters.
  11. Fitness Tracker: An Arduino Nano-based fitness tracker worn as an armband can detect various exercises and count reps for each exercise.
  12. Face Tracking Robot: A robot with a vision sensor tracks and follows a user's face, providing a pet-like experience. It is controlled via an Arduino.
  13. Nerf Turret: A Nerf turret controlled via Bluetooth can be aimed at targets for accuracy testing. It uses servo motors to control pan and tilt movements and shoots darts using small DC motors.
